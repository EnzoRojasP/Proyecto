# Bot de discord, cosas simples pero curiosas

Este proyecto consiste en crear un bot de discord que responda a comandos simples pero curiosos para levantar el ánimo en un servidor de discord
-!Hola = El bot elige un saludo preescrito al azar de una lista y lo dice
-!Hora = El bot da la hora actual 
-!Azar = El bot da un número al azar del 1 al 20
-!P/P/T = Abreviacion de Piedra, Papel o Tijera, el bot dira uno de estos al azar

# Funcionalidades
El proyecto responde a comandos emitidos por un usuario presente dentro de un servidor de discord y enviando una respuesta distinta dependiendo del comando recibido

# Tecnologías
- Python
- Discord
